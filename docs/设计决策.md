# 剪贴板工具 · 设计决策

> 由 `/grill-me` 访谈于 2026-07-03 敲定。原始灵感见 [灵感.md](灵感.md)。本文为实现的源真值。

## 1. 技术栈
Tauri 2 + React + TypeScript + Framer Motion/Motion。
- Rust 后端：剪贴板监听、全局热键、SQLite、托盘、文件存储。
- Web 前端：堆叠卡片、物理动效、搜索、设置。

## 2. 粘贴与焦点模型
- 面板**抢焦点**（IME/中文搜索/方向键原生可用）。
- 打开面板时记录目标窗口 HWND；确认粘贴时 `AttachThreadInput + SetForegroundWindow` 回切前台，再 `SendInput` 模拟 Ctrl+V。
- 粘贴后该条**置顶**。
- 失败兜底：图片贴进纯文本应用 → 静默；回切前台失败 → 内容已写入剪贴板，用户手动 Ctrl+V，面板仍关闭。

## 3. 数据管线
- 监听：`AddClipboardFormatListener` 事件监听，非轮询。
- 文本：纯文本存 SQLite，精确字符串去重，丢弃 HTML/RTF。
- 图片：原图 + 预览(JPEG 最大边 256px、质量 80)存磁盘，blake3 哈希命名 + 精确去重；SQLite 存元数据 + 路径。
- 文件型图片粘贴还原为 HDROP；淘汰条目时清理磁盘文件。
- 粘贴时显式置顶，不依赖监听回环。
- 单条**不设上限**（完整存储）；前端列表只加载前 N 字符预览，完整内容粘贴时按需取。
- 快速连续复制：去抖 200ms，不限速。

## 4. 堆叠面板
- 最新条目在最前（活动卡 = 选中卡）。
- 点击可见卡 = 直接粘贴该卡；Enter = 粘贴最前卡。
- 卡片固定 320×120，图片 contain，文本前 3 行省略。
- 滚轮 / ↓ = 更早，↑ = 更新；可见 3 张（最前全显 + 2 张背后 peek，轻微偏移/缩小/降透明度）。
- 导航弹簧 ~250ms 微动效；到尽头橡皮筋回弹，不循环。
- 关闭：粘贴后关、Esc、点外部、再按 Alt+V。

## 5. 应用生命周期
- 托盘左键 = 开关面板，右键 = 菜单。
- 面板跟随鼠标所在显示器、贴该屏任务栏边沿（适应任务栏位置/自动隐藏）。
- 开机自启默认关、首次引导开启；单实例（Tauri `single-instance` 插件）。

## 6. 常用语与编辑模式
- 顶部 Tab 切换剪贴板/常用语（点击或 Tab 键）。
- 右键 / 长按(600ms) 单卡 = 上下文菜单：
  - 常用语：修改 / 删除 / 移到第一
  - 剪贴板：删除 / 移到第一 / 移入常用语（无修改）
- 仅常用语有「编辑顺序」按钮 → 网格排序模式：面板放大 720×480、卡 200×80、3 列等间距；仅排序（删改回正常视图）；仅鼠标；完成 / Esc 退出。卡片从堆叠「炸开」吸附到网格，带物理惯性 + 弹性缓动；拖拽磁力让位 + 弹簧吸附。
- 新增常用语 = 剪贴板「移入常用语」+ 常用语界面「新建」按钮。
- 无上限；新增放最后（栈底），「移到第一」提到最前；粘贴不重排；仅纯文本。
- ⚠️ 常用语顺序方向与剪贴板相反（新增在最后），有意为之。

## 7. 搜索
- 仅搜剪贴板（常用语不参与搜索）。
- 实时子串、不区分大小写、仅文本（图片搜索时隐藏）。
- 结果切换为扁平列表，点击粘贴；清空 / Esc 回堆叠。
- Alt+C 或点顶部搜索按钮展开搜索框。
- 全量扫描（用户接受超大文本下可能卡顿的代价，50 条上限有界）。

## 8. 主题与设置
- 独立设置窗口。
- light/dark 跟随系统 + 设置手动切；强调色默认蓝、可改预设。
- Alt+V（开关面板）、Alt+C（开搜索）均可在设置自定义，冲突时提示。
- 最大条数默认 50、可调 10–500。
- 设置存 SQLite `settings` 键值表。
- 不做隐私功能（暂停/排除应用/敏感标记跳过）——本地应用，用户选择。

## 9. 边界与性能
- 目标：面板唤起 <100ms；滚动/拖拽 60FPS；后台空闲 RAM <80MB、CPU≈0。
- 超大条目：完整存储 + 全量搜索。
- 粘贴失败：见 §2。
- 排版：系统字体栈，针对 320×120 三行卡调校。

## 已知取舍 / 风险
- 常用语顺序方向与剪贴板相反，打开常用语时最前卡是已置顶/最早的，不是新增的。
- 常用语无搜索功能。
- 搜索全量扫描，多条 MB 级文本时按键可能卡顿。
- 不做隐私功能：复制的密码会进历史并留存直到被挤出。
- 粘贴回切前台偶发失败时降级为手动粘贴。
