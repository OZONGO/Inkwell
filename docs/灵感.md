1，我想使用vibe coding开发一个Windows项目 ，是一个本地运行的剪切板小工具，能在性能控制的很好的情况下做出优秀的视觉效果；
2，点击菜单栏/任务栏常驻图标或快捷键呼出一个浮动面板，在任务栏上方弹出。
3，面板中展示剪切板内容，剪切板内容像手机的堆叠后台那样显示，最下面是第一张盖住第二张和后面的，第一张显示全部内容，第二张只显示未被盖住的，鼠标滚轮或按键盘方向键,上面的卡片盖住下面的。
4，文本卡片用优雅的排版实时渲染，图片用缩略图，两者卡片尺寸统一。
5，使用极简扁平 + 微动效风格。
6，当用户在堆叠中选中某张卡片并确认（鼠标左键单击或键盘enter），卡片内容立刻替换掉当前光标处内容（或粘贴为新内容），面板自动关闭。同时按住 Shift 键粘贴时可以保持面板常驻。并且粘贴后的内容自动变为剪切板第一位。
7， 堆叠卡片最多显示多少张（及是剪切板内容），只保留最近50条，超出后自动淘汰最旧条目。 但用户可设置上限，设计剪切板和常用语功能，用户通过鼠标点击或tab键切换。常用语与剪贴板用同一套视觉语言。且不会像剪切板一样粘贴后的词条自动移动到第一位置。
8，常用语界面中，左键单击卡片（或键盘enter） = 直接粘贴到当前光标处（配合 Shift 保持面板常驻）。
右键单击或长按卡片 = 进入编辑模式（修改内容/删除/移到第一）。
常用语界面有一个按钮，点击可以编辑常用语顺序。
9，进入编辑时，面板本身先稍微放大，在面板内形成成一个规则的矩阵所有卡片完全等大、等间距，暂时抛弃堆叠的 3D 层次，变为纯粹的二维网格用于精准排序，内部卡片“炸开”再吸附到网格位置，带有物理惯性和弹性缓动。
拖拽排序的让位动效拖拽卡片靠近时，周围卡片像被磁力推开，留出空位。松开后，拖拽卡片落入空位，其他卡片弹回新位置。
10，每次复制即写入，后端监听到剪贴板变化，立即同步写入数据库，但判断是否为同样数据，如果新内容在历史中已存在（无论位置），则将该条卡片从原有位置提取到栈顶，不新增条目。
11，打开面板，使用Alt + V或鼠标点击图标，用户在设置可以自定义快捷键。
12，上方常驻搜索按钮，点击按钮后按钮展开成搜索框（使用快捷键alt+c也可以实现同样功能）
13，关闭面板功能，粘贴后关闭 （除非粘贴时按住了shift），Esc 和点击外部都关闭，再次按下alt+v（或用户自定义的快捷键）关闭。
14，剪贴板支持的内容类型范围，仅纯文本 + 图片。如果图片是文件，存压缩版用于快速预览，存路径用于粘贴。如果不是复制文件（截图或网页图片或其他），存压缩版用于快速预览，存原图用于粘贴。
15，剪切板页面和常用语页面一样，可以右键单击或左键长按，进入编辑模式（移入常用语/删除/移到第一）。
16，常用语仅允许存入纯文本。
17，卡片空间里，固定显示前 3 行，超出省略号。
18，实现两套主题，默认跟随系统主题自动切换，设置中用户可以手动切换。
19，必须考虑好边界条件，视觉语言与动画必须统一。
